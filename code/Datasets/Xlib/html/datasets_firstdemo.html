01-MNIST

1.样本为28*28的二值图，数字尺度统一，数字质心在图片正中.

2.训练集6万张图片大约来自250位作者，训练集和测试集中的作者不相交。


02-EMNIST

EMNIST来自NIST Special Database 19，包含了数字和大小写字母。大小为1.65GB。



03-QMNIST

由于MNIST完整的测试集（6万）现在已经无法找到，而且当年MNIST数据集的制作方法也找不到了。 所以提出QMNIST，希望能从MNIST的源头NIST SD-19，生成MNIST数据，尽可能的逼近原始MNIST的预处理效果。

04-KMNIST

Kuzushiji-MNIST。8*28的灰度图，共70k张图片，与MNIST结构相同。包含了10个Hiragana字符。21MB+35KB



05-Fashion MNIST

1.包含了10类，分别为T-shirt/top、Trouser、Pullover、Dress、Coat、Sandal、Shirt、Sneaker、Bag、Ankle boot。
2.与MNIST的数据集划分一样，包含了60k训练集，10k测试集，共70k数据，200MB大小。

06-CIFAR10

1.包含了60k 32*32 的彩色图片，一共10类，每类6k图片，其中50k用于训练，分为5个batch，10k用于测试，分为1个batch。
2.Test batch从10类中的每一类随机采样1k。训练集的每个batch中的各个类的数量不一定相同。

07-USPS

1.与MNIST类似，包含了10个数字类别，训练集7291，测试集2007，共有9298。图片为16*16的灰度图。



08-Omniglot

包含了1623种手写字符，来自50个字符表，每种字符由20个不同的人手写。数据集大小为8MB。




09-PlatVillage

1. 39种不同的植物叶片和背景图像。包含61486个图像

2. 每个图像分辨率为256*256
